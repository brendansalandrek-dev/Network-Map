Purpose
This version uses a full radial layout so each layer has much more space. Parent branches still own their own arc space instead of all nodes being evenly stacked in columns.
How it works
Root book sits at the centre. Child books branch outward in full rings. Drag the map with the mouse to move around. Mouse wheel zooms in and out. Click any node to inspect pages and set its defer value.
